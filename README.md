# 🔥 Raksha Gas Agency Website

A modern, responsive landing page for Raksha Gas Agency - providing fast and reliable LPG delivery services in Bangalore.

## 📋 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Fast Loading**: Optimized for speed and performance
- **SEO Friendly**: Proper meta tags and semantic HTML
- **Accessibility**: WCAG compliant markup
- **Call-to-Action**: Multiple booking options (WhatsApp, Phone)
- **Smooth Animations**: CSS animations and scroll effects
- **Icon Integration**: Font Awesome icons for visual appeal

## 🔧 Customization

You can easily customize:

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #ef4444;      /* Red */
    --secondary-color: #1f2937;    /* Dark Gray */
    --light-bg: #f9fafb;           /* Light Gray */
    --text-dark: #111827;          /* Dark Text */
    --text-light: #6b7280;         /* Light Gray Text */
}
```

### Content
- Update text in `index.html`
- Modify prices and services
- Change phone numbers and WhatsApp link

### Images
- Add your logo
- Update favicon
- Add service images
